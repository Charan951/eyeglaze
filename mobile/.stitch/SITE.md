# EyeGlaze Mobile — Site Vision

**Project:** EyeGlaze Mobile  
**Stitch Project ID:** 3181050102536854073  
**Platform:** Mobile (390px width, Flutter)  
**Brand:** EyeGlaze — Premium Eyewear E-Commerce

## 1. Vision

EyeGlaze is an Indian premium eyewear e-commerce mobile app. The visual identity is dark luxury — ink-black backgrounds with antique gold accents, creating a boutique shopping experience. Every screen should feel like browsing a curated collection in an upscale optical store.

Key user flows:
- Browse → Select → Customize Lens → Add to Cart → Checkout
- Login → Browse → Wishlist → Purchase
- Account management, prescriptions, memberships

## 2. Target Audience

Young Indian professionals (18-35) who value premium eyewear brands, prefer mobile shopping, and appreciate luxury aesthetics in their shopping apps.

## 3. Brand Guidelines

- **Logo:** "EYEGLAZE" in uppercase, gold (#C9A84C), extra-bold, letter-spacing 4px. Subtitle "PREMIUM EYEWEAR" below in gold, lighter weight, letter-spacing 6px.
- **Tone:** Confident, luxury, modern. No playful or casual elements.
- **Imagery:** High-quality product photography, model shots, clean backgrounds.
